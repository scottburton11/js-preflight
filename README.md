js-preflight
============
Run jslint and jsmin on all .js files sourced in html files provided as arguments.
for details: 
    preflight --help

Prerequisites
-------------
Spidermonkey - `brew install spidermonkey`

Bugs
----
Output isn't perfect yet; you'll need to move the packed .js file yourself for now
